This page contains links to projects I've created while taking EDTECH 502.

Project List

  • Netiquette web page: Rules of etiquette for online communications.
  • Accessibility web page: Links to resources pertaining to website design for people with disabilities.
  • Copyright scavenger hunt web page: Learning activity with questions and linked resources to teach fundamental issues of copyright and fair use.
  • Mobile learning web page: A learning activity designed for use with mobile phones or other hand-held devices.
  • Virtual field trip: A virtual tour using multiple Web pages and multimedia to guide learners through an online visit to a remote destination.
  • WebQuest: A higher-order thinking activity that integrates online resources. Multiple Web pages and a CSS menu are used to create this project.
